# CoverGrab — YouTube обложка в 1 клик

Минимальный статический проект на TypeScript для скачивания превью YouTube‑видео.
Пользователь вставляет ссылку, нажимает кнопку — приложение подбирает лучшее доступное качество (maxres → sd → hq → mq → default) и запускает скачивание.

## Возможности

- Поддержка ссылок: `youtube.com/watch?v=…`, `youtu.be/…`, `shorts/…`, `embed/…` и просто `videoId`.
- Автоматический выбор лучшего качества превью с fallback.
- Скачивание через Blob + fallback на новую вкладку при ограничениях CORS.
- Готово для GitHub Pages (статические файлы).

## Быстрый старт

1. Установите зависимости:

```bash
npm install
```

2. Соберите фронтенд:

```bash
npm run build
```

3. Откройте `index.html` напрямую или запустите локальный сервер:

```bash
python3 -m http.server
```

## Деплой на GitHub Pages

1. Запушьте репозиторий в `main`.
2. В GitHub перейдите **Settings → Pages**.
3. Выберите **Deploy from a branch**.
4. Branch: `main`, Folder: `/ (root)`.
5. Сайт станет доступен по адресу: `https://<user>.github.io/<repo>/`.

> Важно: файл `assets/main.js` должен быть собран и закоммичен перед деплоем.

## Как это работает

- Из ссылки извлекается `videoId`.
- Список кандидатов сортируется от максимального качества к минимальному.
- Проверяется реальный размер картинки — так отбрасываются заглушки.
- При успешной загрузке изображение автоматически скачивается.
